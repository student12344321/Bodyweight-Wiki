1. PALM RAISES (ดันพื้นด้วยข้อนิ้ว)

ประเภท: Floor Leverage | เน้น: ปลายแขนหน้า (Forearm Flexors)

ท่าพื้นฐานที่ดีที่สุดในการสร้างกล้ามเนื้อท้องแขนโดยไม่ต้องใช้อุปกรณ์ ใช้พื้นเป็นตัวต้านแรง

ดูบน YouTube →
  • **วิธีฝึก:** คุกเข่า วางมือราบกับพื้น กางนิ้วออก ยก "ฝ่ามือ" ลอยขึ้นโดยที่นิ้วทั้ง 4 ยังติดพื้น แล้ววางลงช้าๆ
  • **เทคนิค:** แขนต้องเหยียดตรงตลอดเวลา อย่างอศอก แรงจะลงที่ท้องแขนเต็มๆ ยิ่งโน้มตัวไปข้างหน้า ยิ่งหนัก
  • **เป้าหมาย:** 3 เซ็ต x 12-15 ครั้ง

2. FINGERTIP PUSH UP (วิดพื้นด้วยปลายนิ้ว)

ประเภท: Isometric Grip | เน้น: ความแข็งแรงของนิ้วและเอ็นมือ

ท่าระดับตำนานที่สร้างแรงบีบ (Grip Strength) และความแข็งแกร่งของเอ็นนิ้วมือ ซึ่งส่งผลโดยตรงต่อขนาดของปลายแขน

ดูบน YouTube →
  • **วิธีฝึก:** ตั้งท่า Push Up แต่ใช้ "ปลายนิ้ว" ยันพื้นแทนฝ่ามือ (ทำมือเป็นรูปถ้วย)
  • **Progression:** เริ่มจากคุกเข่าก่อน หรือแค่เกร็งค้างไว้ (Plank Hold) โดยไม่ต้องวิดพื้นลงไป
  • **เป้าหมาย:** 3 เซ็ต x ค้างไว้ 15-30 วินาที

3. WRIST PUSH UP (วิดพื้นหลังมือ)

ประเภท: Mobility & Strength | เน้น: ปลายแขนหลัง (Extensors)

ท่าที่นักยิมนาสติกใช้เพื่อเสริมสร้างความแข็งแกร่งและยืดหยุ่นให้ข้อมือ (ต้องระวัง ทำช้าๆ)

ดูบน YouTube →
  • **วิธีฝึก:** คุกเข่า วาง "หลังมือ" ลงกับพื้น (นิ้วชี้เข้าหากัน) กดน้ำหนักเบาๆ แล้วพยายามกำหมัดหรือดันข้อมือขึ้น
  • **ข้อควรระวัง:** เริ่มทำบนพื้นนุ่มๆ และใช้เข่ารับน้ำหนักเกือบทั้งหมด ห้ามลงน้ำหนักเต็มที่ทันที
  • **เป้าหมาย:** 3 เซ็ต x 8-10 ครั้ง

4. KNUCKLE PUSH UP (วิดพื้นกำหมัด)

ประเภท: Wrist Stability | เน้น: ความมั่นคงของข้อมือ

การวิดพื้นด้วยหมัดช่วยสร้างความแข็งแรงให้แนวข้อมือตรงและแน่น (Wrist Alignment) ซึ่งสำคัญมากต่อการออกแรง

ดูบน YouTube →
  • **วิธีฝึก:** กำหมัดแล้ววางลงพื้นโดยให้ข้อนิ้วชี้และนิ้วกลางรับน้ำหนัก (แนวเดียวกับกระดูกแขน)
  • **ประโยชน์:** ช่วยให้ข้อมือแข็งแรง ไม่หักงอเวลาออกแรงดันหนักๆ
  • **เป้าหมาย:** 3 เซ็ต x 10-12 ครั้ง

5. SELF-RESISTED WRIST CURL (ใช้มือต้านมือ)

ประเภท: Isolation | เน้น: โฟกัสกล้ามเนื้อเฉพาะจุด

วิธีฝึกที่ง่ายที่สุดและไม่ต้องใช้อุปกรณ์ใดๆ เลย คือการใช้มืออีกข้างสร้างแรงต้านให้ตัวเอง

ดูบน YouTube →
  • **วิธีฝึก:** ยื่นแขนมาข้างหน้า หงายมือขึ้น ใช้มืออีกข้างกดทับฝ่ามือไว้ พยายามงอข้อมือเข้าหาตัวสู้กับแรงกดจากมืออีกข้าง
  • **เทคนิค:** ออกแรงต้านให้หนักพอจนข้อมือเคลื่อนที่ช้าๆ (Slow motion) จะรู้สึกปวดตึงที่ท้องแขนทันที
  • **เป้าหมาย:** 3 เซ็ต x 15-20 ครั้งต่อข้าง
