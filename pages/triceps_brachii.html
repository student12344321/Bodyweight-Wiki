1. DIAMOND PUSH UP (วิดพื้นรูปเพชร)

ประเภท: Compound Push | เน้น: หลังแขนทุกส่วน (Overall Mass)

ราชาแห่งท่าหลังแขน จากผลวิจัย EMG พบว่าเป็นท่าที่กระตุ้นกล้ามเนื้อ Triceps ได้มากที่สุด มากกว่าท่าอื่นๆ ทั้งหมด

ดูบน YouTube →
  • **วิธีฝึก:** วางมือชิดกันให้นิ้วโป้งและชี้ชนกันเป็นรูปเพชร วางมือไว้กลางอก
  • **เทคนิค:** หนีบศอกให้ชิดลำตัวตลอดเวลาขณะลงและขึ้น อย่ากางศอกออก
  • **เป้าหมาย:** 3 เซ็ต x 8-12 ครั้ง

2. FLOOR TRICEPS EXTENSION (สคัลครัชเชอร์บนพื้น)

ประเภท: Isolation | เน้น: หลังแขนหัวยาว (Long Head)

ท่าที่เลียนแบบท่า Skull Crusher ในยิม แต่ใช้พื้นแทน ช่วยสร้างความหนาให้หลังแขนส่วนที่ใหญ่ที่สุด

ดูบน YouTube →
  • **วิธีฝึก:** ตั้งท่า Plank (แขนเหยียด) แล้วค่อยๆ งอศอกลดตัวลงจนศอกแตะพื้น แล้วดันตัวกลับขึ้นด้วยแรงหลังแขน
  • **เทคนิค:** ลำตัวต้องตรงเป็นแผ่นกระดานตลอดเวลา ห้ามงอสะโพก
  • **เป้าหมาย:** 3 เซ็ต x 8-10 ครั้ง

3. CHAIR DIP (ดิปส์กับเก้าอี้)

ประเภท: Compound | เน้น: หลังแขนหัวข้าง & หัวใน (Lateral & Medial)

ท่าคลาสสิกที่ทำได้ง่ายๆ ที่บ้าน ช่วยเพิ่มขนาดและความแข็งแรงให้หลังแขนได้ดี

ดูบน YouTube →
  • **วิธีฝึก:** นั่งขอบเก้าอี้ วางมือข้างลำตัว เลื่อนก้นออกมา แล้วงอศอกหย่อนตัวลงจนศอกทำมุม 90 องศา
  • **ความปลอดภัย:** หลังต้องชิดขอบเก้าอี้ตลอดเวลา อย่าตัวห่างเก้าอี้เพราะจะเจ็บไหล่
  • **เป้าหมาย:** 3 เซ็ต x 10-15 ครั้ง

4. SPHINX PUSH UP (ดันพื้นท่าสฟิงซ์)

ประเภท: Dynamic Plank | เน้น: หลังแขน & แกนกลาง

คล้ายกับท่า Triceps Extension แต่เริ่มจากท่านอนราบ (ศอกวางพื้น) แล้วดันตัวขึ้น เป็นการฝึกแรงระเบิดจากจุดหยุดนิ่ง

ดูบน YouTube →
  • **วิธีฝึก:** เริ่มในท่า Plank (ศอกวางพื้น) มือยื่นไปข้างหน้าเล็กน้อย แล้วออกแรงกดฝ่ามือดันศอกให้ลอยขึ้นจนแขนตึง
  • **เทคนิค:** เกร็งหน้าท้องแน่นๆ อย่าให้หลังแอ่นตอนดันขึ้น
  • **เป้าหมาย:** 3 เซ็ต x 6-10 ครั้ง

5. COBRA PUSH UP (วิดพื้นท่าโควบรา)

ประเภท: Isolation / Finisher | เน้น: บีบหลังแขน (Contraction)

ท่าปิดท้ายที่ดีมาก ไม่เน้นน้ำหนักเยอะ แต่เน้นการ "บีบ" กล้ามเนื้อหลังแขนให้แน่นที่สุดในจุดสูงสุด

ดูบน YouTube →
  • **วิธีฝึก:** นอนคว่ำ วางมือไว้ใต้ไหล่ (เหมือนวิดพื้นปกติ) แต่ให้ "สะโพกติดพื้น" ตลอดเวลา ดันแค่อกขึ้นจนแขนตึง
  • **โฟกัส:** จังหวะแขนตึง ให้เกร็งหลังแขนค้างไว้ 1-2 วินาที
  • **เป้าหมาย:** 3 เซ็ต x 15-20 ครั้ง
