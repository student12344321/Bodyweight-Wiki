1. THE PERFECT PUSH UP (วิดพื้นมาตรฐาน)

ประเภท: Compound Movement | เน้น: อกกลาง & ไหล่หน้า

ท่าพื้นฐานที่ดีที่สุดในการสร้างกล้ามเนื้ออกโดยรวม การวางมือที่ถูกต้องจะช่วยป้องกันการบาดเจ็บที่ไหล่และข้อมือ

ดูบน YouTube →
  • **การวางมือ:** วางมือกว้างกว่าหัวไหล่เล็กน้อย นิ้วชี้ชี้ไปข้างหน้าหรือเฉียงออกเล็กน้อย
  • **ข้อศอก:** ศอกต้องทำมุมประมาณ 45 องศากับลำตัว (ไม่กางออก 90 องศา)
  • **การเกร็ง:** เกร็งก้นและหน้าท้องตลอดเวลา ตัวต้องตรงเป็นไม้กระดาน
  • **เป้าหมาย:** 3-4 เซ็ต x 10-15 ครั้ง

2. WIDE GRIP PUSH UP (วิดพื้นมือกว้าง)

ประเภท: Isolation Bias | เน้น: อกด้านนอก (Outer Chest)

การวางมือที่กว้างขึ้นจะลดการทำงานของแขน (Triceps) และเพิ่มแรงต้านไปที่กล้ามเนื้อหน้าอกโดยตรง

ดูบน YouTube →
  • **วิธีฝึก:** วางมือให้กว้างกว่าท่าปกติประมาณ 1-2 ฝ่ามือ
  • **จุดโฟกัส:** จินตนาการว่ากำลัง "บีบพื้น" เข้าหากันในจังหวะดันตัวขึ้น
  • **เป้าหมาย:** 3 เซ็ต x 8-12 ครั้ง

3. DIAMOND PUSH UP (วิดพื้นรูปเพชร)

ประเภท: Triceps & Inner Chest | เน้น: อกด้านใน & หลังแขน

ท่านี้เน้นหลังแขน (Triceps) อย่างหนัก แต่ก็ช่วยปั้นอกด้านในได้ดีเยี่ยมเมื่อบีบแขนเข้าหากัน

ดูบน YouTube →
  • **วิธีฝึก:** วางมือชิดกันให้นิ้วโป้งและนิ้วชี้ชนกันเป็นรูปเพชร
  • **เทคนิค:** พยายามหนีบศอกให้ชิดลำตัวขณะลง จะโดนหลังแขนเต็มๆ
  • **เป้าหมาย:** 3 เซ็ต x 8-12 ครั้ง

4. DECLINE PUSH UP (วิดพื้นขาอยู่สูง)

ประเภท: Upper Chest Bias | เน้น: อกบน (Upper Chest)

การยกขาให้สูงขึ้นจะเปลี่ยนจุดศูนย์ถ่วงไปที่ "อกส่วนบน" ซึ่งเป็นส่วนที่มักจะเติมเต็มยากที่สุด

ดูบน YouTube →
  • **วิธีฝึก:** พาดเท้าไว้บนเก้าอี้ เตียง หรือบันได แล้ววิดพื้นตามปกติ
  • **ความสูง:** ยิ่งขาอยู่สูง ยิ่งโดนไหล่มากขึ้น ให้หาระดับที่พอดีที่รู้สึกตึงที่อกบน
  • **เป้าหมาย:** 3 เซ็ต x 10-15 ครั้ง

5. PSEUDO PLANCHE PUSH UP (วิดพื้นโน้มตัว)

ประเภท: Strength / Planche Prep | เน้น: ไหล่หน้า & อก (High Intensity)

ท่าปราบเซียนที่ไม่ต้องใช้อุปกรณ์ แต่สร้างแรงต้านได้มหาศาลโดยการ "เปลี่ยนจุดศูนย์ถ่วง" ยิ่งโน้มตัวไปข้างหน้ามากเท่าไหร่ ยิ่งหนักเหมือนยกเวท

ดูบน YouTube →
  • **การวางมือ:** หันนิ้วออกด้านข้างหรือชี้ไปด้านหลัง และวางมือให้อยู่ระดับ "เอว"
  • **เทคนิค:** โน้มตัวไปข้างหน้าให้ **หัวไหล่เลยข้อมือ** ตลอดเวลาทั้งตอนขึ้นและลง
  • **ประโยชน์:** สร้างกล้ามเนื้ออกและไหล่ได้แน่นมาก ทดแทนท่า Dips ได้ดีที่สุดบนพื้นราบ
  • **เป้าหมาย:** 3 เซ็ต x 5-10 ครั้ง (เน้นฟอร์ม ไม่เน้นจำนวน)

6. ARCHER PUSH UP (วิดพื้นแบบนักธนู)

ประเภท: Advanced / Unilateral | เน้น: ความแข็งแรงสูงสุด

ท่าขั้นสูงที่เตรียมร่างกายไปสู่ One Arm Push Up โดยการถ่ายน้ำหนักไปที่แขนข้างเดียว

ดูบน YouTube →
  • **วิธีฝึก:** กางมือให้กว้างมาก ลงน้ำหนักไปที่แขนข้างหนึ่ง อีกข้างเหยียดตึงเพื่อประคอง
  • **เทคนิค:** แขนข้างที่เหยียดต้องตึงตลอดเวลา ไม่ใช่งอช่วยดัน
  • **เป้าหมาย:** 3 เซ็ต x 5-8 ครั้งต่อข้าง
