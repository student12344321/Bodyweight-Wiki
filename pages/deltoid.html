1. DOORWAY REVERSE FLY (กางแขนย้อนหลังวงกบประตู)

ประเภท: Isolation Pull | เน้น: ไหล่หลัง (Rear Delt) โดยตรง

ท่าที่ดีที่สุดในการแยกฝึกไหล่หลังโดยไม่ใช้น้ำหนัก โดยการใช้วงกบประตูเป็นตัวจับแล้วดึงตัวออกด้านข้าง

ดูบน YouTube →
  • **วิธีฝึก:** ยืนที่ขอบประตู จับขอบประตูด้วยมือข้างเดียว (หรือสองข้างถ้าประตูแคบ) เอนตัวไปด้านหลังเล็กน้อย แล้วดึงตัวขึ้นโดยการ "กางแขน" ออก ไม่ใช่ดึงศอก
  • **เทคนิค:** ล็อคศอกให้งอเล็กน้อยคงที่ตลอดเวลา เพื่อให้แรงลงที่ไหล่หลัง ไม่ใช่หลังแขน
  • **เป้าหมาย:** 3 เซ็ต x 12-15 ครั้งต่อข้าง

2. SUPINE ELBOW PUSH (นอนหงายดันศอก)

ประเภท: Pure Isolation | เน้น: ไหล่หลัง 100% (ตัดการช่วยจากหลัง)

ท่ามหัศจรรย์ที่ดูเหมือนง่ายแต่โดนไหล่หลังเต็มๆ โดยการนอนราบและใช้ศอกดันพื้นเพื่อยกตัวขึ้น

ดูบน YouTube →
  • **วิธีฝึก:** นอนหงาย กางแขนออก 90 องศา ตั้งศอกขึ้นกำหมัดชี้ฟ้า กดศอกลงพื้นแรงๆ จนแผ่นหลังช่วงบนลอยขึ้นจากพื้น
  • **ข้อควรระวัง:** อย่าเกร็งหน้าท้องช่วยยกตัว ให้ใช้แรงกดจากไหล่ล้วนๆ
  • **เป้าหมาย:** 3 เซ็ต x 10-15 ครั้ง (เกร็งค้างที่จุดสูงสุด 1 วินาที)

3. PRONE T-RAISE (นอนคว่ำยกแขนตัว T นิ้วโป้งลง)

ประเภท: Floor Isolation | เน้น: ไหล่หลัง & บ่ากลาง

การหมุนนิ้วโป้งลง (Internal Rotation) จะช่วยล็อคการทำงานของไหล่หลังให้ทำงานหนักขึ้นกว่าปกติ

ดูบน YouTube →
  • **วิธีฝึก:** นอนคว่ำ กางแขนออกเป็นตัว T หมุนนิ้วโป้งชี้ลงพื้น (Pinkies Up) แล้วยกแขนขึ้นสูงสุดเท่าที่ทำได้
  • **เทคนิค:** บีบสะบักเข้าหากันทุกครั้งที่ยกแขนขึ้น
  • **เป้าหมาย:** 3 เซ็ต x 15-20 ครั้ง

4. REAR DELT WALL PRESS (ยืนพิงผนังดันศอก)

ประเภท: Isometric / Static | เน้น: ความแข็งแรงไหล่หลังแบบเกร็งค้าง

ท่าฝึกแบบ Isometric ที่ทำได้ทุกที่ เพียงแค่มีผนัง ช่วยกระตุ้นการทำงานของเส้นประสาทกล้ามเนื้อไหล่หลัง

ดูบน YouTube →
  • **วิธีฝึก:** ยืนหันหลังให้ผนัง ห่างออกมาเล็กน้อย กางแขนออก เอาศอกยันผนังไว้ แล้วออกแรงดันผนังเพื่อดันตัวมาข้างหน้า
  • **ความรู้สึก:** จะรู้สึกเกร็งที่หลังไหล่อย่างมาก โดยที่ตัวไม่ต้องขยับเยอะ
  • **เป้าหมาย:** 3 เซ็ต x เกร็งค้าง 20-30 วินาที

5. SIDE LYING REAR DELT FLY (นอนตะแคงกางแขน)

ประเภท: Unilateral Isolation | เน้น: ไหล่หลัง (Range of Motion)

การนอนตะแคงช่วยเพิ่มระยะการเคลื่อนไหว (ROM) ให้ไหล่หลังได้ยืดสุดและหดสุด

ดูบน YouTube →
  • **วิธีฝึก:** นอนตะแคง ยื่นแขนข้างที่อยู่ด้านบนออกไปข้างหน้า แล้ววาดแขนขึ้นไปด้านหลัง (เปิดอก) ช้าๆ ต้านแรงโน้มถ่วง
  • **Tip:** ท่านี้ถ้าไม่มีดัมเบล ให้เน้นทำช้าๆ และเกร็งต้านเอง หรือหาขวดน้ำเล็กๆ มาถือจะดีมาก
  • **เป้าหมาย:** 3 เซ็ต x 15-20 ครั้งต่อข้าง
