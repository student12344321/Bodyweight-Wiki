1. DOORFRAME CURL (ดึงข้อวงกบประตู)

ประเภท: Compound Pull | เน้น: หน้าแขน & หลัง

ท่าพื้นฐานที่ดีที่สุดสำหรับคนไม่มีบาร์โหน ใช้แค่วงกบประตูในการสร้างแรงต้าน ยิ่งวางเท้าเข้าไปลึก ยิ่งหนักขึ้น

ดูบน YouTube →
  • **วิธีฝึก:** ยืนที่วงกบประตู จับขอบประตูระดับอก วางเท้าเข้าไปด้านหน้าแล้วเอนตัวไปด้านหลัง
  • **เทคนิค:** ดึงตัวเข้าหาประตูโดยใช้แรงจากแขนเป็นหลัก (พยายามอย่าใช้หลังช่วยมากเกินไป)
  • **เป้าหมาย:** 3 เซ็ต x 10-12 ครั้ง (ข้างละ)

2. LEG RESISTED CURL (ใช้ขาต้านแรง)

ประเภท: Isolation | เน้น: หน้าแขน (Peak Contraction)

ท่าแยกฝึกหน้าแขนโดยใช้น้ำหนักขาตัวเอง หรือใช้ผ้าขนหนูคล้องใต้ขาเพื่อสร้างแรงต้าน

ดูบน YouTube →
  • **วิธีฝึก:** นั่งบนเก้าอี้หรือพื้น ใช้มือสอดใต้ข้อพับขา (หรือใช้ผ้าคล้อง) แล้วยกขาขึ้นด้วยแรงแขน
  • **เทคนิค:** เกร็งขาต้านแรงแขนไว้ตลอดเวลาเพื่อให้แขนออกแรงเต็มที่
  • **เป้าหมาย:** 3 เซ็ต x 10-15 ครั้งต่อข้าง

3. REVERSE HAND PUSH UP (วิดพื้นกลับมือ)

ประเภท: Compound Push | เน้น: หน้าแขน (Biceps Bias)

การหันนิ้วมือชี้ไปด้านหลัง (หาเท้า) จะเปลี่ยนจุดหมุน ทำให้กล้ามเนื้อหน้าแขนต้องทำงานหนักเพื่อพยุงตัว

ดูบน YouTube →
  • **วิธีฝึก:** วางมือระดับเอว หันนิ้วชี้ไปด้านหลัง (หาเท้า) แล้ววิดพื้น
  • **จุดสำคัญ:** หนีบศอกแนบเลําตัว จะรู้สึกตึงที่หน้าแขนมากกว่าวิดพื้นปกติ
  • **เป้าหมาย:** 3 เซ็ต x 8-12 ครั้ง

4. TOWEL HAMMER CURL (ค้อนผ้าขนหนู)

ประเภท: Isometric / Grip | เน้น: แขนด้านข้าง (Brachialis) & ปลายแขน

ท่าที่ปลอดภัยและทำง่ายที่สุดในการสร้างความกว้างให้แขน (Brachialis) โดยการจับแบบค้อน (Hammer Grip)

ดูบน YouTube →
  • **วิธีฝึก:** คล้องผ้าขนหนูไว้ที่เสาหรือลูกบิดประตู จับปลายผ้าแบบหันฝ่ามือเข้าหากัน
  • **การออกแรง:** ยืนเท้าชิด ดึงศอกแนบเลําตัว แล้วออกแรงดึงผ้าเข้าหาตัวให้แรงที่สุดเท่าที่จะทำได้
  • **เป้าหมาย:** 3 เซ็ต x เกร็งค้างไว้ (Max Effort) 10-15 วินาที

5. TABLE CHIN UP (ดึงข้อใต้โต๊ะ)

ประเภท: Compound Pull | เน้น: หลัง & หน้าแขน

ถ้าที่บ้านมีโต๊ะที่แข็งแรง ท่านี้คือท่าที่ดีที่สุดรองจากการโหนบาร์ เพื่อสร้างกล้ามเนื้อแขนและหลัง

ดูบน YouTube →
  • **วิธีฝึก:** นอนหงายใต้ขอบโต๊ะ จับขอบโต๊ะแบบหงายมือ (Underhand) แล้วดึงอกขึ้นหาขอบโต๊ะ
  • **เทคนิค:** การหงายมือจับจะช่วยให้หน้าแขนทำงานหนักกว่าการคว่ำมือ
  • **เป้าหมาย:** 3 เซ็ต x 8-12 ครั้ง
