1. DOORFRAME ROW (ดึงข้อวงกบประตู)

ประเภท: Compound Pull | เน้น: ปีก & ความหนาของหลัง

ท่าพื้นฐานที่ดีที่สุดสำหรับคนไม่มีอุปกรณ์ เพียงแค่ใช้วงกบประตูที่แข็งแรงก็สร้างหลังกว้างได้

ดูบน YouTube →
  • **วิธีฝึก:** ยืนที่วงกบประตู จับขอบให้มั่นคง วางเท้าเข้าไปด้านหน้า เอนตัวไปข้างหลัง แล้วดึงอกเข้าหาประตู
  • **เทคนิค:** หนีบศอกแนบเลําตัวขณะดึง เพื่อเน้นกล้ามเนื้อปีก (Lats) มากกว่าไหล่หลัง
  • **เป้าหมาย:** 3 เซ็ต x 10-15 ครั้ง

2. HUMAN PULLOVER (สไลด์พื้นดึงปีก)

ประเภท: Vertical Pull Pattern | เน้น: ปีก (ความกว้าง)

ท่าจำลองเครื่อง Lat Pulldown โดยการนอนหงายแล้วดึงตัวเลื่อนไปกับพื้น (ต้องใช้พื้นลื่นๆ)

ดูบน YouTube →
  • **วิธีฝึก:** นอนหงายบนพื้นลื่น (ใส่ถุงเท้าหรือหาผ้ารองหลัง) จับขาโต๊ะหรือขอบประตูไว้ แล้วเกร็งปีกดึงตัวขึ้นไปหาจุดจับ
  • **ทางเลือก:** ถ้าพื้นไม่ลื่น ให้ทำท่านอนคว่ำแล้วใช้มือตะกุยพื้นดึงตัวไปข้างหน้าแทน
  • **เป้าหมาย:** 3 เซ็ต x 8-12 ครั้ง

3. SINGLE ARM DOOR ROW (ดึงประตูมือเดียว)

ประเภท: Unilateral Pull | เน้น: ปีก (ระยะยืดสูงสุด)

การฝึกทีละข้างช่วยให้เราบิดลำตัวและยืดกล้ามเนื้อปีก (Stretch) ได้มากกว่าการทำสองข้างพร้อมกัน

ดูบน YouTube →
  • **วิธีฝึก:** จับขอบประตูมือเดียว เอนตัวไปหลัง ปล่อยตัวให้ไหล่ถูกดึงไปข้างหน้า (ยืดปีก) แล้วดึงศอกกลับมาข้างเอว
  • **ความรู้สึก:** ต้องรู้สึกตึงที่ข้างลำตัวมากที่สุดตอนปล่อยแขนสุด
  • **เป้าหมาย:** 3 เซ็ต x 10-12 ครั้งต่อข้าง

4. PRONE SUPERMAN PULL (นอนคว่ำดึงอากาศ)

ประเภท: Isolation / Squeeze | เน้น: การบีบตัวของปีก

ท่าฝึกโฟกัส (Mind-Muscle Connection) ช่วยให้เรียนรู้วิธีการเกร็งปีกที่ถูกต้องโดยไม่ต้องใช้น้ำหนัก

ดูบน YouTube →
  • **วิธีฝึก:** นอนคว่ำ ยกอกขึ้น ยื่นมือไปข้างหน้า แล้วดึงศอกลงมาข้างลำตัวอย่างช้าๆ พร้อมบีบสะบักและปีก
  • **เทคนิค:** เกร็งค้างที่จังหวะดึงสุด 2 วินาที จินตนาการว่ากำลังบีบมะนาวด้วยรักแร้
  • **เป้าหมาย:** 3 เซ็ต x 15-20 ครั้ง

5. BACK WIDOW (นอนหงายดันศอก)

ประเภท: Isolation Push | เน้น: ปีก & หลังส่วนบน

ท่าดันพื้นกลับด้านที่ใช้ "ศอก" ขับเคลื่อน ช่วยกระตุ้นปีกและหลังกลางได้ดีมากโดยไม่ต้องใช้อุปกรณ์

ดูบน YouTube →
  • **วิธีฝึก:** นอนหงาย วางศอกข้างลำตัว (มุม 45 องศา) กดศอกลงพื้นเพื่อยกอกและหลังลอยขึ้น
  • **โฟกัส:** พยายามใช้แรงจากปีกดันตัวขึ้น ไม่ใช่แรงจากหน้าท้อง
  • **เป้าหมาย:** 3 เซ็ต x 10-15 ครั้ง
