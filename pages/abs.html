1. HOLLOW BODY HOLD (ท่าเรือใบ)

ประเภท: Isometric Core | เน้น: หน้าท้องส่วนลึก & ความมั่นคง (Foundation)

ท่าหัวใจสำคัญของ Calisthenics และยิมนาสติก ช่วยสร้างแรงเกร็งหน้าท้องที่ถูกต้องและป้องกันอาการปวดหลัง

ดูบน YouTube →
  • **วิธีฝึก:** นอนหงาย กดหลังส่วนล่างให้ติดพื้น (ห้ามมีช่องว่าง) ยกไหล่และขาขึ้นลอยจากพื้น
  • **เทคนิค:** ถ้าหลังแอ่น ให้งอเข่าเข้ามาจนกว่าหลังจะแนบพื้นสนิท นี่คือกุญแจสำคัญ
  • **เป้าหมาย:** 3 เซ็ต x ค้างไว้ 20-45 วินาที

2. BICYCLE CRUNCH (ปั่นจักรยานอากาศ)

ประเภท: Dynamic Rotation | เน้น: ท้องด้านข้าง (Obliques) & ท้องบน

งานวิจัย EMG ยกให้เป็นท่าบริหารหน้าท้องที่ดีที่สุดอันดับต้นๆ เพราะมีการบิดตัวและยกขาพร้อมกัน

ดูบน YouTube →
  • **วิธีฝึก:** นอนหงาย มือแตะศีรษะ บิดตัวนำศอกขวาไปแตะเข่าซ้าย สลับข้างไปมา
  • **เทคนิค:** ทำช้าๆ เน้นการบิดเอว ไม่ใช่แค่ขยับศอกไปมา
  • **เป้าหมาย:** 3 เซ็ต x 15-20 ครั้งต่อข้าง

3. LYING LEG RAISE (นอนยกขา)

ประเภท: Hip Flexion | เน้น: ท้องล่าง (Lower Abs)

ท่าคลาสสิกสำหรับจัดการพุงล่าง แต่ต้องทำถูกวิธีเพื่อไม่ให้ปวดหลัง

ดูบน YouTube →
  • **วิธีฝึก:** นอนราบ มือวางข้างลำตัวหรือรองใต้ก้น ยกขาคู่ขึ้นจนตั้งฉาก แล้ววางลงช้าๆ ห้ามให้ส้นเท้าแตะพื้น
  • **ข้อควรระวัง:** กดหลังให้ติดพื้นตลอดเวลา ถ้ารู้สึกปวดหลัง ให้งอเข่าเล็กน้อย
  • **เป้าหมาย:** 3 เซ็ต x 10-15 ครั้ง

4. RUSSIAN TWIST (บิดตัวรัสเซียน)

ประเภท: Anti-Rotation / Dynamic | เน้น: เอว & กล้ามเนื้อข้างลำตัว

ท่าสร้างกล้ามเนื้อ V-Shape และความแข็งแรงในการหมุนตัว

ดูบน YouTube →
  • **วิธีฝึก:** นั่งชันเข่า เอนตัวไปด้านหลังเล็กน้อยจนรู้สึกเกร็งท้อง ยกเท้าลอย (หรือวางส้นเท้า) แล้วบิดมือไปแตะพื้นซ้ายขวา
  • **เทคนิค:** ตามองตามมือที่บิดไป เพื่อให้เกิดการหมุนที่กระดูกสันหลังจริงๆ
  • **เป้าหมาย:** 3 เซ็ต x 20-30 ครั้ง (รวมซ้ายขวา)

5. THE PERFECT PLANK (แพลงก์ที่ถูกต้อง)

ประเภท: Static Stability | เน้น: แกนกลางลำตัวทั้งหมด

การทำ Plank ที่ถูกต้องไม่ใช่แค่การทนค้างไว้ แต่คือการเกร็งทุกส่วนเพื่อสร้างแรงต้านสูงสุด (High Tension Plank)

ดูบน YouTube →
  • **เทคนิค:** ม้วนก้นเข้าหาตัว (Posterior Pelvic Tilt) เกร็งขา เกร็งก้น และดันสะบักขึ้น อย่าให้หลังแอ่น
  • **โฟกัส:** ถ้าทำถูก คุณควรจะสั่นและหมดแรงภายใน 30-60 วินาที ไม่ใช่ค้างได้เป็นนาทีๆ
  • **เป้าหมาย:** 3 เซ็ต x 30-60 วินาที (เน้นคุณภาพ)

6. V-UPS (ท่าตัววี)

ประเภท: Advanced Flexion | เน้น: หน้าท้องโดยรวม (ระเบิดพลัง)

ท่าขั้นสูงที่รวมการทำงานของท้องบนและท้องล่างเข้าด้วยกัน เป็นบททดสอบความแข็งแรงของแกนกลางที่ดีเยี่ยม

ดูบน YouTube →
  • **วิธีฝึก:** นอนหงาย เหยียดแขนขาตรง พับตัวขึ้นมาให้มือแตะปลายเท้าเป็นรูปตัว V
  • **Progression:** ถ้าทำไม่ไหว ให้ทำท่า Tuck Up (งอเข่าเข้าหาอก) แทนไปก่อน
  • **เป้าหมาย:** 3 เซ็ต x 8-12 ครั้ง
